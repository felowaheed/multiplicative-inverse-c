# Multiplicative Inverse Calculator — Extended Euclidean Algorithm

Implementation of the Extended Euclidean Algorithm (EEA) in C to compute the modular multiplicative inverse of a number.

## What it does
Given a number `a` and modulus `m`, calculates `a⁻¹ mod m` — a core operation in RSA and other public-key cryptosystems.

## How to run
gcc multiplicative_inverse.c -o multiplicative_inverse
./multiplicative_inverse
